
# Green Hospitality Hotel Management Frontend

## Tech Stack
- Next.js
- Tailwind CSS
- Lucide Icons

## Features
- Green hospitality theme
- Luxury hotel dashboard
- Reservation management UI
- Housekeeping widgets
- Responsive enterprise layout
- Backend-ready structure

## Installation

```bash
npm install
npm run dev
```

## Recommended Backend
- Django REST Framework
- Node.js API
- FastAPI

## Recommended Deployment
- Netlify
- Vercel
- Render
